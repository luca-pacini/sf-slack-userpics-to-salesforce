# Sanitised Salesforce Metadata

This folder is reserved for Salesforce metadata that has passed a public-safety review.

Only sanitised metadata should be added here.

Do not add:

- Credentials, tokens or secrets.
- Salesforce org identifiers.
- Real usernames or email addresses.
- Internal URLs.
- Exact user mappings.
- Live Named Credential configuration.
- Production deployment scripts.
- Proprietary business rules.

If metadata cannot be safely redacted, keep it out of the public repository.
