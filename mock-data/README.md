# Mock Data

This folder is reserved for mocked public examples only.

Use obvious placeholders such as:

- `MOCK_USER_001`
- `user@example.local`
- `ORG_XXXXX`
- `[MOCKED DATA]`

Do not add real Slack users, Salesforce users, emails, record identifiers, workspace identifiers, org identifiers or payloads copied from production.
